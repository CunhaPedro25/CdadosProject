Anel (Token-Ring)

A topologia lógica do Token Ring é um network Peer to Peer que liga vários Computadores/equipamentos sequencialmente de forma que quando vista num esquema pareca um anel (Ring).

Ex: Estrutura de um Ring

Sistema de Token's

Para a transferencia de dados no Ring circula constantemente um Token a qual quando um peer que queira transmitir dados a "capta" e a modifica para que possa conter informação.

Um pacote de dados, quando modificado para funcionar num Ring, denomina-se de "Frame". Um frame essencialmente é o token, só que contem informação importante para cirular dentro do Ring

Depois do peer ter recebido a transmicao emite um pacote especial denominado por Abort Frame a qual, quando recibida pelo peer que transmite informacao fica suspenso
