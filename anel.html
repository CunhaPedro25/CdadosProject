Anel (Token-Ring)


A topologia lógica do Token Ring é um network Peer to Peer que liga vários Computadores/equipamentos sequencialmente de forma que quando vista num esquema pareca um anel.

Ex: Estrutura de um anel (Ring)

Sistema de Token's

Para a transferência de dados no Anel circula constantemente um Token a qual quando um par que queira transmitir dados a "capta" e a modifica para que possa conter informação.

Um pacote de dados, quando modificado para funcionar num anel, denomina-se de "Frame". Um frame é essencialmente o token, só que contém informação importante para circular dentro do anel

Depois do par ter recebido a transmição emite um pacote especial denominado por Abort Frame a qual, quando recibida pelo par que transmite informação fica suspenso
