Barramento (Bus)


A topologia Lógica do Bus é uma rede descentralizada que liga equipamentos/computadores utilizando uma conexão única para interligar todos os componentes da rede

Ex: Estrutura de um Bus

Sistema de transferencia de Dados

Numa rede de Barramento todos os Pares que estão ligados têm o mesmo acesso a transmição de dados

Para que exista conhecimento de todos os pares na rede existe um Mestre, que normalmente utiliza CSMA (Carrier-Sence Multiple access), que permite a transmição de dados entre os pares na rede

A topologia bus tem problemas com a transferência de dados pois como todos os pares têm a possibilidade de transferir dados ao mesmo tempo existe um possibilidade que aumenta exponencialmente de haver colisão de pacotes.
